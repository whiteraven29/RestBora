# Phase 1: Foundation Tasks

## Objective

Create the base Django project and a working local prototype foundation.

## Tasks

- Create system design docs.
- Scaffold Django project.
- Add the core app.
- Configure installed apps, templates, static files, and REST framework.
- Model the V1 domain.
- Register models in Django admin.
- Add seed data for a demo restaurant.
- Verify migrations and local server startup.

## Done When

The owner can open the dashboard, see demo restaurant data, and navigate to waiter, kitchen, cashier, public menu, inventory, and reports screens.
