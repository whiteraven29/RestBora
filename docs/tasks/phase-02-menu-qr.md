# Phase 2: Menu and QR Setup

## Objective

Make menu setup and QR ordering reliable enough for a real restaurant test.

## Tasks

- Add owner CRUD screens for menu categories.
- Add owner CRUD screens for menu items.
- Add image upload handling for menu items.
- Add table creation and activation/deactivation.
- Generate QR image files or downloadable QR codes per table.
- Add a general restaurant QR menu.
- Improve public menu grouping by category.

## Done When

The owner can manage menu and tables without Django admin, then a customer can scan a table QR and submit an order.
