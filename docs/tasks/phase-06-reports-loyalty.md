# Phase 6: Reports and Loyalty

## Objective

Give the owner simple, useful visibility into performance and repeat customers.

## Tasks

- Add daily, weekly, and monthly profit/loss views.
- Add sales by item.
- Add sales by category.
- Add sales by staff.
- Add customer list and top repeat customers.
- Add manual loyalty adjustment.
- Clearly label profit as estimated while inventory remains manual.

## Done When

The owner can understand sales, expenses, estimated profit, best sellers, staff activity, and loyal customers without technical help.
